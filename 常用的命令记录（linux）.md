
```
ll：查看当前文件夹目录

ls -lA:查看隐藏资源

cd x：进入到文件夹下

ls -l|less：长列表分页查看文件详情（多屏显示通用：空格向下翻页，b向上翻页，q退出）

mkdir x：新建文件夹

git init：git初始化

pwd：显示完整路径

cat x:查看项目目录核心，查看文件里面有什么

cd ~ ；前往用户目录中

vim ：记事本（用来记录东西的）(按i开始编辑，按esc退出，输入“:wq”保存并退出)

:set nu:显示行号

tail -n [numver] [filename]:显示文本的最后n行

git help xxx(具体命令)：查看本地的文档来获取这个命令具体有什么

rm [filename]:删除一个文件

clear : 清空，只留下一个干净的命令框
```
